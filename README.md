## 🔹 分類維度
1. **E / I** → 外向 Extraversion / 內向 Introversion  
2. **S / N** → 感覺 Sensing / 直覺 Intuition  
3. **T / F** → 思考 Thinking / 情感 Feeling  
4. **J / P** → 判斷 Judging / 感知 Perceiving

# 🌈 總結
- **分析者型**：INTJ, INTP, ENTJ, ENTP → 注重邏輯與策略  
- **外交家型**：INFJ, INFP, ENFJ, ENFP → 注重價值與情感  
- **守護者型**：ISTJ, ISFJ, ESTJ, ESFJ → 注重責任與規範  
- **探險家型**：ISTP, ISFP, ESTP, ESFP → 注重行動與自由

# MBTI 個性解析

<details class="category-diplomats">
<summary>🌟 ENFP（競選者 / Campaigner）詳細解析</summary>
<div class="personality-type">

<details>
<summary>🧩 基本特質</summary>
<div class="personality-section">
<strong>E（Extraversion 外向）</strong>：喜歡與人互動，容易受到外界激勵<br>
<strong>N（Intuition 直覺）</strong>：偏好抽象、創意，注重可能性和未來<br>
<strong>F（Feeling 情感）</strong>：重視價值觀與人際關係，關懷他人感受<br>
<strong>P（Perceiving 感知）</strong>：彈性、隨性，享受自由，不喜歡被限制
</div>
</details>

<details>
<summary>🧠 性格核心</summary>
<div class="personality-section">
• 熱情洋溢：對新點子、新體驗和新朋友充滿興趣<br>
• 高度同理心：能快速理解他人情緒並給予支持<br>
• 自由奔放：重視靈活性，抗拒僵化的規則與限制<br>
• 創意驅動：喜歡腦力激盪，從不同角度思考問題<br>
• 理想主義：常常追求有意義的目標，渴望改變世界
</div>
</details>

<details>
<summary>🌍 優勢</summary>
<div class="personality-section">
✅ 有感染力，能帶動氛圍、鼓舞人心<br>
✅ 想像力豐富，能提出創新解決方案<br>
✅ 重視人際，能快速建立信任與關係<br>
✅ 適應力強，面對不確定性仍能積極應對
</div>
</details>

<details>
<summary>⚡ 挑戰</summary>
<div class="personality-section">
⚠ 容易分心：對很多事物有興趣，但難以長期專注<br>
⚠ 害怕常規：不喜歡過於嚴格的制度，可能缺乏紀律<br>
⚠ 過度理想化：對人和事期待過高，失望時受打擊大<br>
⚠ 拖延傾向：因為想做的事情太多，常常無法即時完成
</div>
</details>

<details>
<summary>👥 人際關係</summary>
<div class="personality-section">
<strong>戀愛中</strong>：熱情浪漫、體貼，但需要自由與信任<br>
<strong>友情中</strong>：是團體裡的「開心果」，帶來溫暖和快樂<br>
<strong>家庭中</strong>：關心夢想，重視情感交流，不拘泥傳統角色
</div>
</details>

<details>
<summary>💼 職涯傾向</summary>
<div class="personality-section">
適合表達創意、交流想法、激勵他人的工作：<br>
• 創意產業：設計師、作家、廣告行銷、公關<br>
• 教育/輔導：老師、心理諮商師、培訓師<br>
• 社會影響：非營利組織、社會企業、演講者<br>
• 自由職業：內容創作者、藝術家、創業者
</div>
</details>

<div class="personality-section">
<strong>🌈 座右銘</strong><br>
「生命在於探索與分享熱情」
</div>

</div>
</details>

<details class="category-analysts">
<summary>🧩 INTJ（策劃者 / Architect / Mastermind）詳細解析</summary>
<div class="personality-type">

<details>
<summary>基本特質</summary>
<div class="personality-section">
<strong>I（Introversion 內向）</strong>：偏好獨立思考，從內心世界獲取能量<br>
<strong>N（Intuition 直覺）</strong>：著眼未來，重視抽象概念與長期可能性<br>
<strong>T（Thinking 思考）</strong>：以邏輯、理性和效率做判斷<br>
<strong>J（Judging 判斷）</strong>：有計劃性，追求結構與秩序
</div>
</details>

<details>
<summary>🧠 性格核心</summary>
<div class="personality-section">
• 戰略性思維：擅長全局觀，規劃長遠目標<br>
• 高效率：討厭浪費時間，追求最佳解法<br>
• 獨立自主：依靠自己判斷，不隨波逐流<br>
• 完美主義：對自己和他人要求嚴格<br>
• 冷靜客觀：理性思考，情緒外露少
</div>
</details>

<details>
<summary>🌍 優勢</summary>
<div class="personality-section">
✅ 規劃能力強，能拆解大目標<br>
✅ 高度專注，能長時間投入<br>
✅ 思維縝密，能發現風險與漏洞<br>
✅ 決策果斷，能應對複雜局面
</div>
</details>

<details>
<summary>⚡ 挑戰</summary>
<div class="personality-section">
⚠ 過於挑剔：容易批評低效率的人<br>
⚠ 情感表達弱：可能顯得冷漠<br>
⚠ 過度自信：太相信自己判斷<br>
⚠ 不耐煩：對重複或無效率工作缺乏耐心
</div>
</details>

<details>
<summary>👥 人際關係</summary>
<div class="personality-section">
<strong>戀愛中</strong>：忠誠專一，但內斂，需要被理解<br>
<strong>友情中</strong>：朋友少而精，願意為值得的人付出<br>
<strong>家庭中</strong>：希望家人有目標感，理性規劃型
</div>
</details>

<details>
<summary>💼 職涯傾向</summary>
<div class="personality-section">
適合需要長期規劃、系統思考的職業：<br>
• 科技/工程：軟體工程師、資料科學家、架構師<br>
• 研究/學術：科學家、教授、研究員<br>
• 管理/策略：企業顧問、專案經理、策略規劃師<br>
• 其他：律師、醫生、投資分析師
</div>
</details>

<div class="personality-section">
<strong>🌈 座右銘</strong><br>
「沒有計劃的目標，只是願望」
</div>

</div>
</details>
